## Memorable Date

`<civ-memorable-date>`

Three-field date input (month select + day + year) for dates the user knows from memory. Self-contained. Pass `legend` directly; no external wrapper needed.

#### Spacing — default vs compact

Set `spacing="sm"` to render just the bare input + trigger button with no surrounding label / hint / error chrome. The host's `aria-label` is propagated to the inner control. Used by data-grid cell editors and other dense surfaces where the row or card around the picker already names the field.

<DatePickerProps />

### Calendar Keyboard Navigation

| Key | Action |
|-----|--------|
| Arrow keys | Move by day/week |
| Page Down/Up | Next/previous month |
| Shift + Page Down/Up | Next/previous year |
| Home/End | Start/end of week |
| Enter/Space | Select date |
| T | Jump to today (without selecting) |
| Escape | Close calendar |

### Value Format

ISO `YYYY-MM-DD` regardless of display locale.

### Events

<DatePickerEvents />

### Usage

```html
<civ-date-picker label="Appointment date" hint="Choose an available date" name="appointment" min="2026-01-01" max="2026-12-31" required></civ-date-picker>
```

## Date Range Picker

`<civ-date-range-picker>`

Two linked date pickers that form a start/end range. Cross-validates so end can never precede start. Self-contained. Pass `legend` directly; no external wrapper needed.

### Examples

<StoryEmbed id="forms-inputs-date-range-picker--default" minHeight={450} />
<StoryEmbed id="forms-inputs-date-range-picker--with-outer-bounds" minHeight={450} />
<StoryEmbed id="forms-inputs-date-range-picker--min-range-days" minHeight={450} />
<StoryEmbed id="forms-inputs-date-range-picker--government-leave-request" minHeight={450} />

### Props

<DateRangePickerProps />

### Form Value

Writes two FormData fields: `${name}.start` and `${name}.end` in ISO format.

### Cross-Field Validation

Automatically checks: end ≥ start, duration ≥ min-range-days, duration ≤ max-range-days.

### Events

<DateRangePickerEvents />

### Usage

```html
<civ-date-range-picker
  legend="Leave dates"
  hint="First and last day of leave"
  required
  name="leave"
  min="2026-01-01"
  max="2026-12-31"
  min-range-days="1"
  max-range-days="30"
></civ-date-range-picker>
```
