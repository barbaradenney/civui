`<civ-data-field>`

Displays a label and value as a flat row with an optional inline edit link. Used inside `<civ-summary>` and on review pages to show submitted or prefilled data.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `''` | Field label (left side) |
| `value` | `string` | `''` | Display value (right side, bold) |
| `values` | `string[]` | `[]` | Multiple values, each on its own line (JS property) |
| `edit-href` | `string` | `''` | Edit link destination (renders inline "Edit" link) |
| `edit-label` | `string` | `'Edit'` | Custom edit link text |
| `hint` | `string` | `''` | Optional hint text below the row |
| `spacing` | `'default' \| 'sm'` | `'default'` | Compact variant with smaller padding |

## Usage

```html
<!-- Single value with edit link -->
<civ-data-field
  label="Phone number"
  value="(555) 123-4567"
  edit-href="#/contact/phone"
></civ-data-field>

<!-- Multi-value (address) -->
<civ-data-field
  label="Mailing address"
  .values="${['123 Main St', 'Springfield, IL 62701']}"
  edit-href="#/contact/address"
></civ-data-field>

<!-- With hint -->
<civ-data-field
  label="Email"
  value="jane@agency.gov"
  hint="From your login.gov account"
></civ-data-field>
```
