`<civ-select>`

Accessible dropdown select using a native `<select>` element for maximum accessibility and mobile compatibility. Supports grouped options via `optgroup` and [built-in presets](#presets) for common government data lists.

## Presets

The `preset` prop pre-populates the select with a standard option list. Use `preset-variant` to choose a variant of that list.

| Preset | Variants | Description |
|--------|----------|-------------|
| `us-state` | (default), `territories` | US states + DC. `territories` adds PR, GU, VI, AS, MP. |
| `service-branch` | (default), `all` | Active-duty branches. `all` adds reserve, guard, and historical. |
| `discharge-type` | — | Military discharge types |
| `suffix` | — | Name suffixes (Jr., Sr., II, III, etc.) |
| `relationship-type` | (default), `dependent` | Relationship to veteran |
| `marital-status` | — | Marital status options |
| `ethnicity` | — | OMB ethnicity categories |
| `gender` | (default), `binary` | Gender identity. `binary` limits to Male/Female. |
| `language` | — | Preferred language (Census Bureau frequency) |
| `housing-status` | — | Housing status (HUD/Census categories) |
| `education-level` | — | Educational attainment (Census ACS / ISCED) |
| `employment-status` | — | Employment status (BLS CPS categories) |
| `income-source` | — | Income source (SSA/HUD benefits patterns) |
| `veteran-status` | — | Veteran preference status (OPM 5 USC § 2108) |
| `disability-type` | — | Disability categories (ADA / DOL CC-305) |
| `citizenship-status` | — | Citizenship/work authorization (USCIS I-9) |
| `pay-frequency` | — | Pay frequency (DOL FLSA / IRS Pub 15) |
| `contact-preference` | — | Preferred contact method (USDS/18F patterns) |

### Customizing Presets

Presets are a starting point — you can modify them or provide your own options entirely.

**Override with your own options** (ignore presets):

```html
<civ-select label="Status" name="status">
    <option value="active">Active</option>
    <option value="inactive">Inactive</option>
  </civ-select>
```

**Start from a preset and modify in JavaScript:**

```js
import { resolvePresetOptions } from '@civui/core';

// Get the base preset options
const options = resolvePresetOptions('education-level');

// Add a custom option
options.push({ value: 'trade-certificate', label: 'Trade certificate' });

// Remove an option
const filtered = options.filter(o => o.value !== 'doctoral');

// Apply to the select
document.querySelector('civ-select').options = filtered;
```

The `resolvePresetOptions(preset, variant?)` function is exported from `@civui/core` and returns a plain array of `{ value, label }` objects that you can modify freely.

> **Tip:** Presets also work on `<civ-radio-group>` and `<civ-checkbox-group>` — use the same `preset` and `preset-variant` attributes. This is useful when you want to present a shorter preset list as radio buttons or checkboxes instead of a dropdown.

### Data Sources

All preset data is sourced from federal standards and regulations. Each preset includes a source citation in the source code. Key references:

- **OMB Directive 15** — Race and ethnicity categories
- **Census Bureau ACS** — Marital status, education, language
- **BLS Current Population Survey** — Employment status
- **OPM 5 USC § 2108** — Veteran preference categories
- **ADA / DOL CC-305** — Disability categories
- **USCIS I-9** — Citizenship/work authorization
- **10 USC § 101** — Military service branches

## Usage

```html
<!-- Basic select with slotted options -->
<civ-select label="Branch of service" name="branch" required>
    <option value="army">Army</option>
    <option value="navy">Navy</option>
    <option value="airforce">Air Force</option>
    <option value="marines">Marine Corps</option>
    <option value="coastguard">Coast Guard</option>
    <option value="spaceforce">Space Force</option>
  </civ-select>

<!-- With grouped options -->
<civ-select label="Document type" hint="Choose a document type" name="docType">
    <optgroup label="Military">
      <option value="dd214">DD214</option>
      <option value="dd215">DD215</option>
    </optgroup>
    <optgroup label="Identity">
      <option value="passport">Passport</option>
      <option value="license">Driver's license</option>
    </optgroup>
  </civ-select>

<!-- Using a preset -->
<civ-select label="State" name="state" preset="us-state" required></civ-select>
```
