# Social Security Number

`<civ-ssn>`

Pre-configured input for Social Security numbers with automatic masking, validation, and plain-language labeling. Supports full 9-digit or last-4-only modes.

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "Social Security number" and can be overridden via the `label` prop.

## Component-Specific Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `'full' \| 'last4'` | `'full'` | Whether to collect full 9-digit SSN or last 4 digits only |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- Full SSN with built-in masking and validation -->
<civ-ssn name="ssn" required></civ-ssn>

<!-- Last 4 digits only -->
<civ-ssn name="ssn-last4" mode="last4" required></civ-ssn>
```
