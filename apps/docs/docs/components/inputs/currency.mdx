## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "Amount" and can be overridden via the `label` prop.

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- Currency input with automatic formatting -->
<civ-currency name="amount" required></civ-currency>
```
