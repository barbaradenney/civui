# Phone Link

`<civ-phone-link>`

A link component for telephone numbers. Generates a proper `tel:` href, auto-formats 10-digit US numbers for display (e.g., `8005551234` becomes `(800) 555-1234`), and shows a phone icon. Non-digit characters are stripped for the `tel:` protocol, while the `+` prefix is preserved for international numbers.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `number` | `string` | `''` | Phone number (required). Non-digit characters are stripped for the `tel:` href; `+` is preserved for international numbers. |
| `label` | `string` | `''` | Optional display text override. When omitted, 10-digit US numbers are auto-formatted as `(XXX) XXX-XXXX`. |
| `disabled` | `boolean` | `false` | Disabled state (sets `aria-disabled` and removes href) |

## Why use this component

Malformed `tel:` URLs are a common source of bugs -- spaces, dashes, and parentheses in the href prevent the link from working on mobile devices. This component strips formatting characters automatically so the `tel:` protocol always receives a clean number, while the visible text stays human-readable.

## Usage

```html
<!-- US number: displays as (800) 555-1234, links to tel:8005551234 -->
<civ-phone-link number="8005551234"></civ-phone-link>

<!-- With formatting in the input (stripped automatically) -->
<civ-phone-link number="(800) 555-1234"></civ-phone-link>

<!-- Custom label override -->
<civ-phone-link
  number="8005551234"
  label="Call the VA help desk"
></civ-phone-link>

<!-- International number (+ is preserved) -->
<civ-phone-link number="+44 20 7946 0958"></civ-phone-link>

<!-- TTY number with custom label -->
<civ-phone-link
  number="711"
  label="TTY: 711"
></civ-phone-link>
```
