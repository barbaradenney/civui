# Action Sheet

`<civ-action-sheet>`

A shared popup/overlay component that renders as an absolute dropdown on desktop and a fixed bottom sheet on mobile (≤480px). Use it for any popup content — menus, confirmations, option lists, or custom overlays.

The parent component controls the `open` state. The action sheet fires `civ-action-sheet-close` when it wants to close and the parent decides whether to close.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | `false` | Controls visibility |
| `max-height` | `string` | `'50vh'` | Max height on mobile bottom sheet |
| `trap-focus` | `boolean` | `false` | Enable focus trapping (for modal-like behavior) |
| `no-click-outside` | `boolean` | `false` | Disable click-outside close |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-action-sheet-close` | — | Fires when the sheet wants to close (escape, click outside, backdrop tap) |

## Behavior

### Desktop (>480px)

Renders as an absolute-positioned dropdown below its parent. No backdrop.

### Mobile (≤480px)

Renders as a fixed bottom sheet:
- Full-width, anchored to bottom of viewport
- Rounded top corners with upward shadow
- Semi-transparent backdrop overlay
- Scrollable content with configurable max-height
- Tap backdrop or press Escape to close

### Keyboard

- **Escape** — fires `civ-action-sheet-close`
- **Tab** — cycles through focusable elements (when `trap-focus` is set)

## Usage

```html
<!-- Basic popup menu -->
<div style="position: relative;">
  <civ-button label="Options" @click="${() => this.menuOpen = !this.menuOpen}"></civ-button>
  <civ-action-sheet
    ?open="${this.menuOpen}"
    @civ-action-sheet-close="${() => this.menuOpen = false}"
  >
    <ul>
      <li>Edit</li>
      <li>Duplicate</li>
      <li>Delete</li>
    </ul>
  </civ-action-sheet>
</div>

<!-- Confirmation dialog with focus trap -->
<civ-action-sheet
  ?open="${this.confirmOpen}"
  trap-focus
  @civ-action-sheet-close="${() => this.confirmOpen = false}"
>
  <div class="civ-p-4">
    <p>Are you sure?</p>
    <civ-button label="Confirm" @click="${this.onConfirm}"></civ-button>
    <civ-button variant="secondary" label="Cancel" @click="${() => this.confirmOpen = false}"></civ-button>
  </div>
</civ-action-sheet>
```

## Design principle

All popups, dropdowns, and dialogs in CivUI must use bottom sheets on mobile. The action sheet component enforces this automatically. Components that currently implement their own popup CSS (date picker, combobox) follow the same pattern.
