# Breadcrumb

<!-- contract-link:start -->

> **Contract reference:** [`civ-breadcrumb`](pathname:///civui/storybook/?path=/docs/contract-civ-breadcrumb--docs). Auto-generated structured contract for cross-platform implementation.

<!-- contract-link:end -->

`<civ-breadcrumb>` + `<civ-breadcrumb-item>`

A breadcrumb trail showing the user's path through a hierarchy. The parent renders a `<nav>` landmark containing an `<ol>` of items. The last item should set `current` (or omit `href`). That item renders as a non-link `<span aria-current="page">`. Chevron separators between items are added by CSS so consumers don't have to ship a separator character per item.

## Usage

```html
<civ-breadcrumb>
  <civ-breadcrumb-item href="/" label="Home"></civ-breadcrumb-item>
  <civ-breadcrumb-item href="/benefits" label="Benefits"></civ-breadcrumb-item>
  <civ-breadcrumb-item label="Healthcare" current></civ-breadcrumb-item>
</civ-breadcrumb>
```

The final crumb represents the current page. Either set `current` or omit `href`. Both render the same non-link `<span aria-current="page">` element.

## Accessibility

- Renders as a `<nav>` landmark with `aria-label="Breadcrumb"` by default. Override with the `label` prop when a page has more than one breadcrumb trail.
- The current page uses `aria-current="page"` so screen readers announce the user's position.
- Chevron separators are decorative (rendered via CSS `::before`) and are not part of the accessible name.
