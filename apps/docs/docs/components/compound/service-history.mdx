# Service History

`<civ-service-history>`

Compound component for a single military service period — branch, dates, discharge type, and optional service number. Designed to be used inside a `civ-repeater` for multiple service periods.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `legend` | `string` | `'About this service period'` | Fieldset legend |
| `heading-level` | `1`–`6` | — | Promote the legend to a heading via `role="heading"` + `aria-level=N`. See [Heading hierarchy](/components/form/form-group#heading-hierarchy). |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'sm'` | Visual size. `sm` = body size; `md`/`lg`/`xl` step up for section/page heading use. |
| `name` | `string` | `''` | Form field name prefix |
| `value` | `string` | `''` | JSON-serialized ServicePeriodValue |
| `required` | `boolean` | `false` | Shows required indicator on legend |
| `disabled` | `boolean` | `false` | Disables all fields |
| `show-service-number` | `boolean` | `false` | Show service number field (pre-1974 era) |
| `branch-error` | `string` | `''` | Error for branch of service |
| `start-date-error` | `string` | `''` | Error for service start date |
| `end-date-error` | `string` | `''` | Error for service end date |
| `discharge-error` | `string` | `''` | Error for character of service |
| `service-number-error` | `string` | `''` | Error for service number |

## Branch options

Army, Navy, Air Force, Marine Corps, Coast Guard, Space Force, NOAA, USPHS

## Discharge type options

Honorable, General (under honorable conditions), Other than honorable, Bad conduct, Dishonorable, Undesirable

## Usage

```html
<civ-repeater legend="Service periods" name="service" item-label="service period" mode="form-steps" min="1" max="20">
  <civ-service-history name="period"></civ-service-history>
</civ-repeater>
```
