What It Does

The CivUI MCP Server connects to AI assistants like Claude Desktop via the Model Context Protocol. It exposes a complete toolbox for converting, generating, validating, and operating accessible government forms — from first PDF upload to production-ready React or Web Components output, with workflow UI, email/PDF notices, analytics, i18n, and 508 reports along the way.

How It Works As a Whole

The server is a single McpServer instance that wires three surfaces for the AI client: resources (reference docs the model can read), tools (deterministic functions the model can call), and prompts (multi-step workflows the user can invoke as slash commands). Every tool response automatically appends a _nextSteps suggestion field so the model can chain tools into pipelines without prompting.

1. Resources

11 markdown knowledge sources served over civui://* URIs. The model reads these for ground truth on components, tokens, patterns, schema, and accessibility.

2. Tools

80 Zod-validated functions for parsing, generation, validation, testing, content, utility lookups, workflow UIs, features, and US government forms. Results are JSON with next-step hints.

3. Prompts

9 guided workflows the user can invoke directly. Each prompt orchestrates multiple tool calls behind a single high-level task like “convert a legacy form” or “audit 508 compliance.”

4. Tier Filtering

Set CIV_MCP_TIER to essential (~12 tools), standard (~35), or all (default, 80) to right-size the toolbox for your context window and use case.

5. Next-Step Chaining

Every tool response is post-processed through withNextSteps(), which appends suggested follow-up tools. The AI naturally chains: parse → generate → validate → report.

6. Discovery

The discover_tools tool lets the model query the toolbox itself by tier, category, or keyword — no hard-coded prompt lists to maintain.

Setup

Claude Desktop Configuration

Add this to your Claude Desktop claude_desktop_config.json:

{
  "mcpServers": {
    "civui": {
      "command": "npx",
      "args": ["@civui/mcp-server"]
    }
  }
}

Once configured, Claude Desktop has access to all 80 tools, 11 resources, and 9 guided prompts. To narrow the toolbox for smaller contexts, set CIV_MCP_TIER=essential (~12 tools) or CIV_MCP_TIER=standard (~35). Omit for the full set.
