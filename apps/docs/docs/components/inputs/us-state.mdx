# US State

Use `<civ-select preset="us-state">` for a pre-populated US states select.

Pre-configured select for US states and DC, with optional territory support. Renders a native `<select>` for maximum accessibility.

## Usage

```html
<!-- US states and DC -->
<civ-form-field label="State" required>
  <civ-select name="state" preset="us-state" required></civ-select>
</civ-form-field>

<!-- Include territories -->
<civ-form-field label="State" required>
  <civ-select name="state" preset="us-state" preset-variant="territories" required></civ-select>
</civ-form-field>
```

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

## Preset Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `preset` | `string` | - | Set to `"us-state"` for US states |
| `preset-variant` | `string` | - | Set to `"territories"` to include US territories (PR, GU, VI, AS, MP) |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |
