`<civ-icon>`

An inline SVG icon component using Material Icons Outlined paths. Each icon renders as an `<svg>` element with `fill="currentColor"` — no font files, no external requests.

Icons inherit `color` and scale with `font-size`, so they automatically match surrounding text.

## Built-In Icons

CivUI ships only the 14 icons used by its own components:

- **Navigation:** `chevron-left`, `chevron-right`, `chevron-down`, `external-link`
- **Status:** `check`, `check-circle`, `error`, `warning`, `info`, `loading`
- **Actions:** `close`, `download`
- **Communication:** `mail`, `phone`

For anything outside this set, register your own SVG path with `registerIcon()` or opt in to the [Material Symbols font](../../foundations/icons#beyond-the-built-in-set).

## Accessibility

- When `label` is set, the icon renders with `role="img"` and `aria-label` for screen readers.
- When `label` is omitted, the icon is decorative and renders with `aria-hidden="true"`.
- Always set `label` on icons that convey meaning (e.g., status indicators, action buttons).

## Usage

```html
<!-- Decorative icon (hidden from screen readers) -->
<civ-icon name="check-circle"></civ-icon>

<!-- Meaningful icon with accessible label -->
<civ-icon name="error" label="Error"></civ-icon>

<!-- Sized and colored -->
<civ-icon name="mail" size="lg" class="civ-text-primary"></civ-icon>

<!-- Rotated -->
<civ-icon name="chevron-right" rotate="90"></civ-icon>

<!-- Inline with text -->
<p><civ-icon name="info" size="sm"></civ-icon> Additional information available</p>
```
