# Action Button

`<civ-action-button>` and `<civ-button-group>`

A compact button for toolbars, inline form controls, and secondary actions. Same variant system as `<civ-button>` but with smaller padding and font size. Can be used standalone or grouped inside `<civ-button-group>` to form a connected toolbar.

## civ-action-button Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `''` | Button text |
| `variant` | `'primary' \| 'secondary' \| 'tertiary'` | `'tertiary'` | Visual variant |
| `disabled` | `boolean` | `false` | Disabled state |
| `pressed` | `boolean` | `false` | Toggle pressed state (for toolbar toggles), sets `aria-pressed` |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | HTML button type attribute |

## Variants

- **Primary** -- filled blue, for the main toolbar action
- **Secondary** -- light tint background
- **Tertiary** (default) -- outlined border, for most toolbar buttons

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-analytics` | -- | Analytics tracking event fired on click |

## Usage

```html
<!-- Standalone action button -->
<civ-action-button label="Bold"></civ-action-button>

<!-- Primary variant -->
<civ-action-button label="Save" variant="primary"></civ-action-button>

<!-- Toggle button with pressed state -->
<civ-action-button label="Bold" pressed></civ-action-button>

<!-- Button group (connected toolbar) -->
<civ-button-group>
  <civ-action-button label="Bold"></civ-action-button>
  <civ-action-button label="Italic"></civ-action-button>
  <civ-action-button label="Underline"></civ-action-button>
</civ-button-group>
```
