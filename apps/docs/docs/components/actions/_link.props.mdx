| `href` | `string` | `''` | Link destination. When `type` is `phone` / `email` / `download`, the type-specific props build the href and this prop is ignored. Ignored when `as="button"`. Dangerous protocols (`javascript:`, `data:`, `vbscript:`) are stripped at render time |
| `variant` | `'primary'` \| `'secondary'` \| `'back'` | `'secondary'` | Visual emphasis. `primary` = bold underlined link with trailing caret (most prominent); `secondary` (default) = plain underlined link; `back` = back-navigation link with leading left chevron. | `as` | `'link'` \| `'button'` | `'link'` | Render shape. `link` (default) emits an `<a>` for real navigation; `button` emits a `<button>` styled with link chrome for affordances that read as a link but fire a callback (e.g. inline "Try again" after a fetch error).
