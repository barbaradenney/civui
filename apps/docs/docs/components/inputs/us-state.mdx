# US State

`<civ-us-state>`

Pre-configured select for US states and DC, with optional territory support. Renders a native `<select>` for maximum accessibility.

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "State" and can be overridden via the `label` prop.

## Component-Specific Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `include-territories` | `boolean` | `false` | Include US territories (PR, GU, VI, AS, MP) in the options list |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- US states and DC -->
<civ-us-state name="state" required></civ-us-state>

<!-- Include territories -->
<civ-us-state name="state" include-territories required></civ-us-state>
```
