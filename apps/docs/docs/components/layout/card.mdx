# Card

`<civ-card>`

A structured container with header, body, and footer slot areas. The card provides border, padding, and section styling. Use data attributes to assign children to header and footer; all other content goes into the body.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `'blue' \| 'teal' \| 'red' \| 'green' \| 'yellow' \| 'orange' \| 'purple' \| 'gray' \| ''` | `''` | Color variant (same palette as civ-tag) |
| `card-style` | `'primary' \| 'secondary' \| 'tertiary'` | `'tertiary'` | Emphasis style: primary (dark fill), secondary (light tint), tertiary (white + border) |
| `spacing` | `'default' \| 'sm'` | `'default'` | Padding size inside the card |
| `icon-start` | `string` | `''` | Icon name to render before the body content |
| `icon-end` | `string` | `''` | Icon name to render after the body content |

## Slots

| Slot Attribute | Description |
|----------------|-------------|
| `data-card-header` | Content rendered in the card header area |
| `data-card-footer` | Content rendered in the card footer area |
| _(default)_ | All other children render in the body |

## Usage

### Basic card

```html
<civ-card>
  <p>Card content goes here.</p>
</civ-card>
```

### Card with header and footer

```html
<civ-card>
  <div data-card-header>
    <civ-tag label="In progress" variant="teal"></civ-tag>
    <h3 class="civ-heading-md">Disability compensation</h3>
  </div>
  <p>Filed: March 10, 2026</p>
  <div data-card-footer>
    <civ-link href="/claims/123" variant="secondary">View details</civ-link>
  </div>
</civ-card>
```

### Card styles

```html
<!-- Primary — filled blue, white text -->
<civ-card card-style="primary">
  <p>Important content.</p>
</civ-card>

<!-- Secondary — light tint background -->
<civ-card card-style="secondary">
  <p>Supporting content.</p>
</civ-card>

<!-- Tertiary (default) — white with border -->
<civ-card>
  <p>Standard content.</p>
</civ-card>
```

### Card colors

```html
<!-- Color with secondary style (light tint, default) -->
<civ-card color="green">
  <p>Success message.</p>
</civ-card>

<!-- Color with primary style (dark fill) -->
<civ-card color="red" card-style="primary">
  <p>Error alert.</p>
</civ-card>
```

### Small spacing

```html
<civ-card spacing="sm">
  <p>Compact card content.</p>
</civ-card>
```

### Card with expandable footer

```html
<civ-card>
  <div data-card-header>
    <h3 class="civ-heading-md">Payment history</h3>
  </div>
  <p>Last payment: $1,234.56</p>
  <details data-card-footer>
    <summary>View breakdown</summary>
    <p>Detail content here</p>
  </details>
</civ-card>
```
