# EIN

`<civ-ein>`

Pre-configured input for Employer Identification Numbers with automatic masking, validation, and PII-aware blur-mode masking.

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "Employer Identification Number (EIN)" and can be overridden via the `label` prop.

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- EIN input with built-in masking and validation -->
<civ-ein name="ein" required></civ-ein>
```
