CivUI provides three progress indicator components for multi-step forms:

- **`civ-progress-steps`** — segmented progress bar (one rectangle per step)
- **`civ-progress-header`** — compact "Step X of Y: Title" text header
- **`civ-progress-percent`** — percentage-based progress bar

All three are used by `civ-form-step` via the `progress` prop, or standalone.

## Segmented Progress (`civ-progress-steps`)

Each step is a small rectangle — filled for completed, highlighted for current, and empty for upcoming. Compact at any screen size.

## Using with `civ-form-step`

The `progress` prop on `civ-form-step` selects which indicator to display:

```html
<!-- Compact header (default) -->
<civ-form-step progress="minimal">...</civ-form-step>

<!-- Segmented bar -->
<civ-form-step progress="steps">...</civ-form-step>

<!-- Percentage bar -->
<civ-form-step progress="bar">...</civ-form-step>
```

The back link is rendered consistently above all three indicators and is handled by `civ-form-step`.
