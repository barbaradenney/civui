## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `''` | Link text (preferred over child text) |
| `href` | `string` | `''` | Link destination (required) |
| `variant` | `'primary' \| 'secondary' \| 'tertiary' \| 'back'` | `'tertiary'` | Visual variant |
| `danger` | `boolean` | `false` | Destructive action styling (red text) |
| `disabled` | `boolean` | `false` | Disabled state (sets `aria-disabled` and removes href) |
| `icon-start` | `string` | `''` | Icon name before link text (overrides back variant default chevron-left) |
| `icon-end` | `string` | `''` | Icon name after link text (overrides secondary variant default chevron-right) |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-analytics` | — | Analytics tracking event fired on click |

## Variants

### Primary

Button-styled link with filled blue background and underlined text.

```html
<civ-link href="/apply" variant="primary" label="Start application"></civ-link>
```

### Secondary

Underlined text link with a trailing chevron-right icon.

```html
<civ-link href="/benefits" variant="secondary" label="View all benefits"></civ-link>
```

### Tertiary

Plain underlined link (default variant).

```html
<civ-link href="/privacy" label="Privacy policy"></civ-link>
```

### Back

Back-navigation link with a leading chevron-left icon.

```html
<civ-link href="/previous-step" variant="back" label="Go back"></civ-link>
```

## Danger modifier

Add `danger` to any variant for destructive action styling.

```html
<civ-link href="/delete" variant="secondary" danger label="Remove application"></civ-link>
```

## Disabled state

```html
<civ-link href="/locked" variant="secondary" disabled label="Locked section"></civ-link>
```

## Specialized Links

CivUI provides purpose-built link components for common patterns. Each one wraps the correct protocol, icon, and accessibility announcements so you don't have to assemble them manually.

### External Link

`<civ-external-link>`

A link for navigating to external websites. Automatically sets `target="_blank"`, `rel="noopener noreferrer"`, appends an external-link icon, and adds screen-reader-only "(opens in new tab)" text.

<StoryEmbed id="actions-external-link--default" />

<StoryEmbed id="actions-external-link--disabled" />

[Open in Storybook →](pathname:///civui/storybook/?path=/story/actions-external-link--default)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `''` | Link text |
| `href` | `string` | `''` | External URL destination (required) |
| `disabled` | `boolean` | `false` | Disabled state |

**Why use this component:** Forgetting `rel="noopener noreferrer"` creates a security vulnerability (reverse tabnapping). Forgetting to announce "opens in new tab" disorients screen reader users. This component handles both automatically.

```html
<civ-external-link
  href="https://www.ssa.gov"
  label="Social Security Administration"
></civ-external-link>
```

### Phone Link

`<civ-phone-link>`

A link for telephone numbers. Generates a proper `tel:` href, auto-formats 10-digit US numbers (e.g., `8005551234` becomes `(800) 555-1234`), and shows a phone icon.

<StoryEmbed id="actions-phone-link--default" />

<StoryEmbed id="actions-phone-link--with-label" />

<StoryEmbed id="actions-phone-link--international" />

[Open in Storybook →](pathname:///civui/storybook/?path=/story/actions-phone-link--default)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `number` | `string` | `''` | Phone number (required). Non-digit characters are stripped for `tel:` href; `+` is preserved for international. |
| `label` | `string` | `''` | Display text override. When omitted, 10-digit US numbers auto-format as `(XXX) XXX-XXXX`. |
| `disabled` | `boolean` | `false` | Disabled state |

```html
<civ-phone-link number="8005551234"></civ-phone-link>
<civ-phone-link number="8005551234" label="Call the VA help desk"></civ-phone-link>
<civ-phone-link number="+44 20 7946 0958"></civ-phone-link>
<civ-phone-link number="711" label="TTY: 711"></civ-phone-link>
```

### Download Link

`<civ-download-link>`

A link for file downloads. Sets the `download` attribute, shows a download icon, and optionally displays the file size.

<StoryEmbed id="actions-download-link--default" />

<StoryEmbed id="actions-download-link--with-file-size" />

[Open in Storybook →](pathname:///civui/storybook/?path=/story/actions-download-link--default)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | `''` | URL of the file to download (required) |
| `label` | `string` | `''` | Link text describing the download |
| `filename` | `string` | `''` | Suggested filename for the downloaded file |
| `file-size` | `string` | `''` | Human-readable file size (e.g., "2.4 MB") |
| `disabled` | `boolean` | `false` | Disabled state |

```html
<civ-download-link
  href="/forms/va-10-10ez.pdf"
  label="Application for Health Benefits (VA 10-10EZ)"
  filename="va-10-10ez.pdf"
  file-size="1.2 MB"
></civ-download-link>
```

### Email Link

`<civ-email-link>`

A link for email addresses. Generates a proper `mailto:` href with optional subject line, shows a mail icon, and defaults display text to the address.

<StoryEmbed id="actions-email-link--default" />

<StoryEmbed id="actions-email-link--with-label" />

<StoryEmbed id="actions-email-link--with-subject" />

[Open in Storybook →](pathname:///civui/storybook/?path=/story/actions-email-link--default)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `address` | `string` | `''` | Email address (required) |
| `label` | `string` | `''` | Display text. Defaults to the email address when omitted. |
| `subject` | `string` | `''` | Pre-filled subject line (URL-encoded automatically) |
| `disabled` | `boolean` | `false` | Disabled state |

```html
<civ-email-link address="help@va.gov"></civ-email-link>
<civ-email-link address="support@va.gov" label="Contact VA support"></civ-email-link>
<civ-email-link address="benefits@va.gov" subject="Question about my claim"></civ-email-link>
```
