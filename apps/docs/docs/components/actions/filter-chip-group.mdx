# Filter Chip Group

`<civ-filter-chip-group>`

Coordinates a row of [`civ-filter-chip`](./filter-chip) children. Adds **roving tabindex** so the row is one Tab stop with arrow-key navigation, enforces single-select if `mode="single"`, and emits an aggregated `civ-change` carrying the full selected value(s).

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `'single' \| 'multi'` | `'multi'` | `single` enforces radio-like behavior (one chip selected); `multi` lets each chip toggle independently |
| `label` | `string` | `''` | Accessible label, applied as `aria-label` on the toolbar wrapper |

## Events

| Event | Detail | Fires when |
|-------|--------|------------|
| `civ-change` | `{ value: string \| string[], aggregated: true }` | Any child chip toggles. `value` is a single string in single mode, an array in multi mode. The per-chip `civ-change` is stopped at the group, so listeners attached to the group only receive the aggregate. |

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `value` | `string \| string[]` | Currently selected value(s). Single mode returns a string (or `''` if none); multi mode returns an array. |

## Keyboard

| Key | Action |
|-----|--------|
| `Tab` | Move into / out of the group (one stop, lands on the first selected chip or first chip) |
| `→` / `↓` | Focus next chip |
| `←` / `↑` | Focus previous chip (wraps at the start) |
| `Home` | Focus first chip |
| `End` | Focus last chip |
| `Enter` / `Space` | Toggle the focused chip (handled by the chip itself) |

Disabled chips are skipped during arrow-key navigation.

## Patterns

### Faceted search with multi-select

```html
<civ-filter-chip-group mode="multi" label="Filter results">
  <civ-filter-chip label="Healthcare" value="health" count="24"></civ-filter-chip>
  <civ-filter-chip label="Education" value="education" count="12"></civ-filter-chip>
  <civ-filter-chip label="Housing" value="housing" count="8"></civ-filter-chip>
</civ-filter-chip-group>

<script>
  document.querySelector('civ-filter-chip-group').addEventListener('civ-change', (e) => {
    console.log('selected:', e.detail.value); // ['health', 'housing']
  });
</script>
```

### View mode picker (single-select)

```html
<civ-filter-chip-group mode="single" label="Sort">
  <civ-filter-chip label="Newest" value="newest" selected></civ-filter-chip>
  <civ-filter-chip label="Oldest" value="oldest"></civ-filter-chip>
  <civ-filter-chip label="A → Z" value="alpha"></civ-filter-chip>
</civ-filter-chip-group>
```
