# Image Preview

`<civ-image-preview>`

Displays an uploaded or referenced image at a readable size with optional filename and file size caption. Use for ID document uploads, profile photos, damage photos, signature captures, and review pages.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | `''` | Image source URL or object URL. Required for the component to render. |
| `alt` | `string` | `''` | Alt text for the image. Required for accessibility. |
| `filename` | `string` | `''` | Filename displayed below the image in a `<figcaption>`. |
| `file-size` | `string` | `''` | File size displayed after the filename (e.g., `'2.4 MB'`). |
| `size` | `'sm'` \| `'md'` \| `'lg'` \| `'full'` | `'md'` | Preview size: `sm` (120px), `md` (240px), `lg` (360px), `full` (100%). |

## Usage

```html
<civ-image-preview
  src="/uploads/id-front.jpg"
  alt="Front of driver's license"
  filename="id-front.jpg"
  file-size="2.4 MB"
  size="lg"
></civ-image-preview>
```

## Accessibility

- Renders as a `<figure>` with an `<img>` and optional `<figcaption>`.
- The `alt` prop is required and should describe the image content meaningfully.
- Images use `loading="lazy"` for performance.
