Accessible text input with label, hint, error, and width variants. Supports input masking, declarative validation, prefix/suffix text, and inline icons. Uses ElementInternals for native form participation. [Specialized presets](#specialized-presets) for SSN, phone, email, ZIP, EIN, currency, routing number, and VA file number are documented below.

### Character Counter

When `maxlength` is set, the input renders a "characters remaining" counter below the field. Visual count updates immediately; screen-reader announcements debounce on a 1s interval to avoid spamming. The counter is suppressed when a mask is active (the mask drives the length constraint).

## Custom Mask Patterns

For formats not covered by the specialized components, use `mask-pattern` with slot syntax:

| Slot | Matches |
|------|---------|
| `#` | Digit (0-9) |
| `A` | Letter (a-z, A-Z) |
| `*` | Any printable character |

All other characters are inserted as literals.

### Mask Modes

- **`blur`** (default): User types freely. Mask applied on blur for display. Accessible and predictable.
- **`live`**: Formats as user types. Has accessibility tradeoffs (cursor jumping).

## Specialized Presets

These components extend `<civ-text-input>` with pre-configured masking, validation, labels, hints, and HTML attributes. All share the same events (`civ-input`, `civ-change` with `{ value: string }` detail).

### SSN (`<civ-ssn>`)

Pre-configured input for Social Security numbers with automatic masking, validation, and PII-aware blur-mode masking. Supports full 9-digit or last-4-only modes.

Default label: "Social Security number"

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `'full' \| 'last4'` | `'full'` | Whether to collect full 9-digit SSN or last 4 digits only |

<StoryEmbed id="forms-inputs-text-input--ssn" />

<StoryEmbed id="forms-inputs-text-input--ssn-last-4" />

```html
<civ-ssn label="Social Security number" name="ssn" required></civ-ssn>

<!-- Last 4 digits only -->
<civ-ssn label="Last 4 of Social Security number" name="ssn-last4" mode="last4" required></civ-ssn>
```

### EIN (`<civ-ein>`)

Pre-configured input for Employer Identification Numbers with automatic masking, validation, and PII-aware blur-mode masking.

Default label: "Employer Identification Number (EIN)"

<StoryEmbed id="forms-inputs-text-input--ein" />

```html
<civ-ein label="Employer Identification Number" name="ein" required></civ-ein>
```

### ZIP Code (`<civ-zip>`)

Pre-configured input for US ZIP codes with automatic masking and validation. Supports standard 5-digit and extended ZIP+4 formats.

Default label: "ZIP code"

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `extended` | `boolean` | `false` | Enable ZIP+4 format (e.g., 12345-6789) |

<StoryEmbed id="forms-inputs-text-input--zip-code" />

<StoryEmbed id="forms-inputs-text-input--zip-plus-4" />

```html
<!-- Standard 5-digit ZIP code -->
<civ-zip label="ZIP code" name="zip" required></civ-zip>

<!-- ZIP+4 format -->
<civ-zip label="ZIP code" name="zip" extended required></civ-zip>
```

## Usage

```html
<!-- Basic text input -->
<civ-text-input label="Full name" name="fullName" required></civ-text-input>

<!-- Custom mask pattern -->
<civ-text-input label="Case number" hint="For example: ABC-1234" name="caseNumber" mask-pattern="AAA-####"></civ-text-input>

<!-- With prefix -->
<civ-text-input label="Social handle" name="handle" prefix="@" width="md"></civ-text-input>

<!-- With suffix -->
<civ-text-input label="Weight" name="weight" suffix="lbs" inputmode="decimal" width="sm"></civ-text-input>

<!-- With character counter -->
<civ-text-input label="Short bio" name="bio" maxlength="60"></civ-text-input>
```
