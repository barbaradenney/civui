# ZIP Code

`<civ-zip>`

Pre-configured input for US ZIP codes with automatic masking and validation. Supports standard 5-digit and extended ZIP+4 formats.

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "ZIP code" and can be overridden via the `label` prop.

## Component-Specific Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `extended` | `boolean` | `false` | Enable ZIP+4 format (e.g., 12345-6789) |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- Standard 5-digit ZIP code -->
<civ-zip name="zip" required></civ-zip>

<!-- ZIP+4 format -->
<civ-zip name="zip" extended required></civ-zip>
```
