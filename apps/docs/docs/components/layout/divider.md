# civ-divider

A horizontal rule for visually separating content sections. Renders as an `<hr>` with consistent styling from design tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `spacing` | `'default' \| 'sm'` | `'default'` | Vertical spacing around the divider |

## Examples

### Default spacing

```html
<civ-divider></civ-divider>
```

### Small spacing

```html
<civ-divider spacing="sm"></civ-divider>
```
