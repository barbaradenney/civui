MCP Server

AI-powered form modernization. Convert legacy PDF and HTML government forms into accessible, Section 508 compliant CivUI web components.

83 Tools 10 Resources 8 Guided Prompts Section 508

What It Does

The CivUI MCP Server connects to AI assistants like Claude Desktop, providing tools that parse legacy government forms (PDF, HTML), convert them into structured FormSchema definitions, and generate accessible CivUI web components. Every generated form is automatically validated against Section 508 accessibility standards.

Tool Categories

Parsing

  • parse_html_form — Extract fields from HTML forms

  • generate_civui_form — Generate accessible CivUI markup

  • generate_tests — Vitest test suite for forms

  • check_contrast — Color contrast ratio checking

  • export_schema — Export FormSchema as JSON

Setup

Claude Desktop Configuration

Add this to your Claude Desktop claude_desktop_config.json:

{
  "mcpServers": {
    "civui": {
      "command": "npx",
      "args": ["@civui/mcp-server"]
    }
  }
}

Once configured, Claude Desktop will have access to all 83 tools, 10 resources, and 8 guided prompts for converting legacy forms into accessible CivUI components.
