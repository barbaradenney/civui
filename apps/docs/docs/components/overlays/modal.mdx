`<civ-modal>`

A general-purpose modal dialog. Centered overlay on desktop, bottom sheet on mobile (≤480px). Includes focus trap, escape-to-close, optional backdrop close, and scroll support for tall content.

## Behavior

- **Desktop:** Centered overlay, max-width 600px, max-height 80vh
- **Mobile (≤480px):** Bottom sheet with rounded top corners, max-height 85vh
- Focus is trapped inside the modal (native `<dialog>` behavior)
- Body scroll is locked when open, restored on close
- Focus returns to the trigger element on close
- Native `<dialog>` provides implicit `role="dialog"` and modal semantics
- Use `label` prop for accessible name when no heading is set
- **Escape** closes by default — use `no-escape-close` to prevent
- **Required decisions:** Combine `no-close-button`, `no-backdrop-close`, and `no-escape-close` to force the user to pick an action

## Usage

```html
<!-- Standard modal with footer actions -->
<civ-modal ?open="${this.showModal}" heading="Confirm"
  @civ-modal-close="${() => this.showModal = false}">
  <p>Are you sure you want to continue?</p>
  <div data-modal-footer>
    <civ-button variant="secondary" label="Cancel"
      @click="${() => this.showModal = false}"></civ-button>
    <civ-button label="Continue"
      @click="${this.onConfirm}"></civ-button>
  </div>
</civ-modal>

<!-- Required decision — no escape or backdrop close -->
<civ-modal ?open="${this.showConfirm}" heading="Delete application"
  no-close-button no-backdrop-close>
  <p>This action cannot be undone.</p>
  <div data-modal-footer>
    <civ-button variant="secondary" label="Cancel"
      @click="${() => this.showConfirm = false}"></civ-button>
    <civ-button danger label="Delete"
      @click="${this.onDelete}"></civ-button>
  </div>
</civ-modal>
```
