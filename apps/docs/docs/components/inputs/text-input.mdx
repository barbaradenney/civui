# Text Input

`<civ-text-input>`

Accessible text input with label, hint, error, and width variants. Supports input masking, declarative validation, prefix/suffix text, and inline icons. Uses ElementInternals for native form participation.

:::tip Use specialized components for common field types
For SSN, phone, email, ZIP, EIN, currency, routing number, and VA file number, use the dedicated components (`<civ-ssn>`, `<civ-phone>`, etc.) instead of configuring `<civ-text-input>` manually. They pre-configure masking, validation, labels, hints, and HTML attributes automatically.
:::

### Character Counter

When `maxlength` is set, the input renders a "characters remaining" counter below the field. Visual count updates immediately; screen-reader announcements debounce on a 1s interval to avoid spamming. The counter is suppressed when a mask is active (the mask drives the length constraint).

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`.

## Custom Mask Patterns

For formats not covered by the specialized components, use `mask-pattern` with slot syntax:

| Slot | Matches |
|------|---------|
| `#` | Digit (0-9) |
| `A` | Letter (a-z, A-Z) |
| `*` | Any printable character |

All other characters are inserted as literals.

### Mask Modes

- **`blur`** (default): User types freely. Mask applied on blur for display. Accessible and predictable.
- **`live`**: Formats as user types. Has accessibility tradeoffs (cursor jumping).

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- Basic text input -->
<civ-text-input
  label="Full name"
  name="fullName"
  required
></civ-text-input>

<!-- Custom mask pattern -->
<civ-text-input
  label="Case number"
  name="caseNumber"
  mask-pattern="AAA-####"
  hint="For example: ABC-1234"
></civ-text-input>

<!-- With prefix -->
<civ-text-input
  label="Social handle"
  name="handle"
  prefix="@"
  width="md"
></civ-text-input>

<!-- With suffix -->
<civ-text-input
  label="Weight"
  name="weight"
  suffix="lbs"
  inputmode="decimal"
  width="sm"
></civ-text-input>

<!-- With character counter -->
<civ-text-input
  label="Short bio"
  name="bio"
  maxlength="60"
></civ-text-input>
```
