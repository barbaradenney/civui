# Form Step

`<civ-form-step>`

Multi-step navigation within a form chapter. Shows one step at a time, validates required fields before advancing, and renders a progress indicator with a back link.

Each direct child element with a `data-step-label` attribute is treated as a step. Only the current step is visible.

## Progress Modes

The `progress` prop selects which indicator to display. All three include a step header and consistent back link.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `progress` | `'minimal' \| 'steps' \| 'bar'` | `'minimal'` | Progress indicator style |
| `header-size` | `'primary' \| 'secondary' \| 'tertiary'` | `'secondary'` | Step header size |
| `heading-level` | `number` | `2` | Semantic heading level (2–6) |
| `step-title` | `string` | `''` | Title override for the current step |
| `persist` | `string` | `''` | Storage key for auto-saving form data via `civ-form` |
| `continue-label` | `string` | `'Continue'` | Label for the continue button |
| `complete-label` | `string` | `'Save'` | Label for the final step's button |
| `validate` | `boolean` | `true` | Enable built-in required field validation before advancing |
| `nav-disabled` | `boolean` | `false` | Disable navigation buttons |
| `hide-nav` | `boolean` | `false` | Hide all navigation UI |
| `sensitive` | `boolean` | `false` | Marks the step as emotionally sensitive |
| `show-pause` | `boolean` | `false` | Render a "Save and come back later" action |
| `pause-label` | `string` | `''` | Override the pause action label |
| `beforeContinue` | `(stepEl, index) => Promise<boolean> \| boolean` | — | Async validation callback (JS property only) |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-step-change` | `{ current, total, label }` | When the step changes |
| `civ-step-back` | `{ from, to }` | When back is clicked |
| `civ-step-continue` | `{ from, to }` | When continue is clicked and validation passes |
| `civ-step-complete` | `{ total }` | When the last step is completed |
| `civ-step-pause` | `{ current, label }` | When pause is clicked |

## Usage

```html
<!-- Default (minimal header) -->
<civ-form-step>
  <div data-step-label="Your name">...</div>
  <div data-step-label="Date of birth">...</div>
</civ-form-step>

<!-- With segmented progress bar -->
<civ-form-step progress="steps">
  <div data-step-label="Eligibility">...</div>
  <div data-step-label="Personal Info">...</div>
  <div data-step-label="Review">...</div>
</civ-form-step>

<!-- With percentage bar and tertiary header -->
<civ-form-step progress="bar" header-size="tertiary">
  <div data-step-label="Step 1">...</div>
  <div data-step-label="Step 2">...</div>
</civ-form-step>

<!-- Primary header with divider lines -->
<civ-form-step header-size="primary">
  <div data-step-label="Personal Info">...</div>
  <div data-step-label="Review">...</div>
</civ-form-step>
```
