# Country

`<civ-country>`

Pre-configured select for countries with US listed first by default. Supports filtering by inclusion or exclusion lists.

## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "Country" and can be overridden via the `label` prop.

## Component-Specific Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `us-first` | `boolean` | `true` | List the United States first in the dropdown |
| `include` | `string` | — | Comma-separated list of country codes to include (e.g., `"US,CA,MX"`) |
| `exclude` | `string` | — | Comma-separated list of country codes to exclude |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- Country select with US listed first -->
<civ-country name="country" required></civ-country>

<!-- North America only -->
<civ-country name="country" include="US,CA,MX" required></civ-country>
```
