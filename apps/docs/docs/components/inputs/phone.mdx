## Shared Props

All form components share these props: `label`, `name`, `value`, `hint`, `error`, `required`, `disabled`, `readonly`.

The label defaults to "Phone number" and can be overridden via the `label` prop.

## Component-Specific Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `international` | `boolean` | `false` | Enable international phone number format |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `civ-input` | `{ value: string }` | Fires on every value change |
| `civ-change` | `{ value: string }` | Fires on committed value change |

## Usage

```html
<!-- US phone number with automatic formatting -->
<civ-phone name="phone" required></civ-phone>

<!-- International phone number -->
<civ-phone name="phone" international required></civ-phone>
```
